# **ADR-0003: One Primary Concept per Document**

**Status**

Accepted

## **Context**

Large documents become difficult for both humans and AI.

## **Decision**

Every AIR document should represent one primary concept.

Related concepts should be linked using document references.

## **Consequences**

Advantages

- Smaller documents
- Better AI context retrieval
- Easier navigation

Tradeoffs

- Increased document count
- Greater reliance on links
